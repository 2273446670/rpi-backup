# rpi-backup

树莓派系统备份脚本

1、back.sh（备份脚本）
   用法：./back.sh xxx.img  (xxx.img为备份文件名，自行修改)
   
2、resize.sh（扩容脚本）
   用法：恢复系统后执行 ./resize.sh 扩容root分区，也可以用树莓派raspi-config进行扩容
